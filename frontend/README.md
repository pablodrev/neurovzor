# Frontend - Neurovzor (Med-Viewer)

Фронтенд медицинской диагностической платформы для анализа рентген-снимков. Приложение позволяет загружать DICOM/PNG/JPG изображения и проводить интерактивный анализ патологий в режиме реального времени.

## 🎯 Описание проекта

**Neurovzor** — это современное веб-приложение для медицинской диагностики, которое:

- ✅ Загружает и визуализирует медицинские изображения (DICOM, PNG, JPG)
- ✅ Предоставляет интерактивные инструменты для анализа рентген-снимков
- ✅ Поддерживает разные роли пользователей (врачи и студенты)
- ✅ Использует передовые ML-модели для автоматизированного анализа
- ✅ Имеет отзывчивый интерфейс с современной дизайн-системой

### Текущий функционал

- **Страница загрузки** — UploadPage для проверки и отправки изображений
- **Просмотровщик врача** — DoctorViewer для анализа и диагностики
- **Просмотровщик студента** — StudentViewer для обучения

## 🛠️ Технологический стек

- **React 18** — UI-фреймворк
- **Vite** — быстрый бандлер и dev-сервер
- **Radix UI** — компоненты низкого уровня для доступности
- **SCSS** — препроцессор CSS с переменными и миксинами
- **Cornerstone.js** — работа с медицинскими изображениями (DICOM)
- **Lucide React** — иконки
- **class-variance-authority** — типобезопасные CSS-классы

## 📦 Установка

### Предварительные требования

- Node.js 16+ 
- npm или yarn

### Шаги установки

```bash
# Перейти в папку фронтенда
cd frontend

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev

# Собрать продакшн-версию
npm run build

# Проверить код ESLint
npm run lint

# Предпросмотр собранного приложения
npm run preview
```


## 🎯 Основные страницы

### UploadPage
Позволяет пользователям загружать медицинские изображения (DICOM, PNG, JPG) для анализа.

### DoctorViewer
Предоставляет полный набор инструментов для врачей:
- Просмотр изображений
- Интерактивные замеры и разметка
- Результаты анализа ML-моделей
- Экспорт отчетов

### StudentViewer
Образовательный интерфейс для студентов:
- Просмотр примеров диагностики
- Интерактивное обучение
- Демонстрация правильной техники анализа

## 🔌 Интеграция с Backend

Фронтенд взаимодействует с Backend API (FastAPI) через REST запросы:

```javascript
// Пример загрузки изображения
const formData = new FormData();
formData.append('file', imageFile);

fetch('/api/hip_dysplasia/analyze', {
  method: 'POST',
  body: formData
});
```

Backend находится в папке `../backend/` на одном уровне.

## 📝 Соглашения по коду

- Имена компонентов — PascalCase (например, `DoctorViewer.jsx`)
- Имена переменных — camelCase (например, `isLoading`)
- SCSS файлы начинаются с нижнего подчеркивания (например, `_mixins.scss`)
- Каждый компонент находится в отдельной папке с файлами `.jsx` и `.scss`

## 🐛 Отладка

### Cornerstone.js

Для работы с DICOM-изображениями используется Cornerstone.js. Если изображения не загружаются, проверьте:
- CORS-заголовки на backend
- Формат файла (DICOM должен быть валидным)
- Консоль браузера на наличие ошибок
